# 餐廳清單
此餐廳清單可以搜尋餐廳、點擊餐廳圖案以取得更多資訊

## 功能列表
- 使用者可以透過搜尋餐廳名稱來找到特定的餐廳
- 使用者可以透過點擊餐廳圖案以取得更多資訊
- 使用者可以透過點擊地址旁的圖案得到 google maps 的相關資料

## 安裝
- 開啟終端機(Terminal)cd 到存放專案本機位置並執行:
git clone https://github.com/jtuuang/restaurant_list.git
- 初始
npm install
- 開啟程式
npm run dev
